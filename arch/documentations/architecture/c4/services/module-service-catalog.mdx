# üíÖ Service Catalog Module

This module manages all service offerings and pricing for the My Clients platform.

## Responsibilities

- Manage service definitions (name, description, duration)
- Set and update service pricing
- Support service packages and bundles
- Track service popularity and usage
- Provide service data to scheduling and analytics modules

## Key Components

- **Service Repository:** Handles CRUD operations for service data in the database
- **Service Catalog Service:** Business logic for managing services, pricing, and packages
- **API Endpoints:** REST endpoints for service management (create, update, list, etc.)
- **Integration:** Interacts with Schedule and Analytics modules

## Data Model

- Service ID
- Name, description, duration
- Price (with support for dynamic pricing)
- Package/bundle info
- Popularity metrics

## Interactions

- **Schedule Module:** Provides available services for appointment booking
- **Analytics Module:** Supplies service data for performance and revenue analysis

## Technologies

- PHP (Symfony)
- PostgreSQL (service data storage)
- REST API
